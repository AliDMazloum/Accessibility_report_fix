================================================================================
TECHNICAL NOTES
================================================================================

Slide Index Mapping:
  - Image filenames use 1-indexed slide numbers (slide4_shape3.png)
  - Python API uses 0-indexed slides {'slide': 3, 'shape_idx': 3}
  - Script automatically converts between the two

File Modification:
  - Alt texts stored as XML attributes in PPTX (zip archive)
  - Not visible in slide content, only in metadata
  - Accessible via right-click "Edit Alt Text" in PowerPoint
  - Compatible with all screen readers (NVDA, JAWS, etc.)

Performance:
  - Typical runtime: 30-60 seconds for all 5 files
  - File size impact: <100KB for all 57 alt texts
  - No network access required
  - All operations are local and reversible
